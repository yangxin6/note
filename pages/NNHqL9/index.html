# 点云MLP

交换输入顺序：

交换点云的输入顺序，并不改变点云本身的数据，但是对神经网络确是不同的输入，从而产生了不同的输出，这就对神经网络很不友好。

# PointNet（PN）

  • 点云中的每个点单独处理

  • 池化层的函数特点：函数必须要保证 改变点的输入顺序 不改变输出结果，例如：max mean等

  • T-Net：给输入点云做了一个旋转

  • 输入点云 做归一化

    • 就是我们想模拟的函数

  • MAX 函数： 输入 n 个向量，给出逐元素最大值

谱聚类

谱聚类：在图的层面工作，而不是欧氏距离，关注点与点之间的连接性。在不规则的情况下可以选择谱聚类。

# 构建无向图

  • 每个点都是一个节点
  • 有3种常用的为谱聚类构建无向图的方法
  1. -neighborhood Graph. Radius 邻近搜索。离一个点多远范围内的点都建立连线，
  2. k-neighborhood Graph. kNN搜索。每个节点直选离它最近的 k 个节点建立连线，
    • 如果 是 的其中一个 knn 或 是 的其中一个 knn，则建立连线
    • 如果 是 的其中一个 knn 且 是 的其中一个 knn，则建立连线
  3. 每一个节点之间都有连线，连线的权重 就是 相似度（距离的导数）

# 区别

未归一化的谱聚类：更倾向于 每一个类里面的数据是均等的。

已归一化的谱聚类：更倾向于 每个类里面的粒度是接近的。

通常归一化谱聚类在点密度不均匀的情况下效果很好。

# 总结

  • 复杂度：

    • k-means 的复杂度

  • 优点：

    • 不会对每个类的形状有 任何的假设
    • 工作在图论上(连接性)，而不是欧几里得距离,
    • 可以对任何维度的数据操作
    • 自动发掘多少个类

  • 缺点：

    • 时间复杂度较高

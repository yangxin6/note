# 步骤

  1. 初始化
  2. E-step. 计算潜在变量Z的概率 （即GMM中的后验概率 ）
  3. M-step. 计算
    • （$E_Z $ 关于的期望）
  4. 检查是否收敛，如果不收敛，返回E-step.

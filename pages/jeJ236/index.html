# 优化方法：

  • Gradient descent（梯度下降）-- 一次下降
  • Gauss-Newton（高斯牛顿）-- 二次下降

  • 简单、快速

大量噪声、数据中有很多模型 这样的用 Hough Transform 和 Random Sample Consensus(RANSAC)
